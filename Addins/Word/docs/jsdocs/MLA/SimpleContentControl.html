MLA.SimpleContentControl

MLA.SimpleContentControl()
    Create a new SimpleContentControl instance.

MLA.SimpleContentControl(id)
    Create a new SimpleContentControl instance setting the id for the control. The intended usage is for the id to be set to the ContentControl.ID property that identifies the control as it exists in the active document.
SimpleContentControl Properties

MLA.SimpleContentControl.id
    The id property of the content control in the active document.

MLA.SimpleContentControl.tag
    The tag property of the content control in the active document.

MLA.SimpleContentControl.title
    The title property of the content control in the active document.

MLA.SimpleContentControl.type
    The type of content control.

    A content control can be any of the following types:

    wdContentControlBuildingBlockGallery
    wdContentControlPicture
    wdContentControlComboBox
    wdContentControlDropdownList
    wdContentControlDate
    wdContentControlGroup
    wdContentControlText
    wdContentControlRichText

MLA.SimpleContentControl.lockcontrol
    A String value of "True" or "False"

    If True, the control can not be deleted from the Active Document. Contents can still be edited.

MLA.SimpleContentControl.lockcontents
    A String value of "True" or "False"

    If True, the contents can not be edited in the Active Document. The control can still be deleted.

MLA.SimpleContentControl.parentID
    The id property of the parent content control for this SimpleContentControl.

    This property will only be available if the content control is embedded within another content control in the active document.

    Content controls can only be nested within rich-text content controls, building block gallery content controls, and group content controls.

MLA.SimpleContentControl.parentTag
    The tag property of the parent content control for this SimpleContentControl.

    This property will only be available if the content control is embedded within another content control in the active document.

    Content controls can only be nested within rich-text content controls, building block gallery content controls, and group content controls.
