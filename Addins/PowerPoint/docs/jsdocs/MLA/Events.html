   +--MLA.<events>
Event Handling


Events are captured in the PowerPoint application and information is passed to MarkLogicPowerPointEventSupport.js. To insure framework code is not tightly coupled with application code, developers should define their custom event handler functions in a separate .js file and call from MarkLogicPowerPointEventSupport.js.

Following events captured:

colorSchemeChange()
newPresentation()
slideSelectionChange()
slideShowBegin()
slideShowEnd()
slideShowNextBuild()
slideShowNextClick()
slideShowNextSlide()
presentationBeforeSave()
presentationClose()
presentationNewSlide()
presentationOpen()
presentationPrint()
presentationSave()
windowActivate()
windowDeactivate()

Event Detail

colorSchemeChange(slideRangeName)
    Occurs after a color scheme is changed. Returns name of slideRange where color scheme change occurred.
newPresentation(presentationName)
    Occurs after a presentation is created. Returns the name of the new presentation.
slideSelectionChange(slideIndex)
    Occurs at different times depending on the current view. Returns index of slide selected.
slideShowBegin(presentationName)
    Occurs when you start a slide show. Returns name of presentation showing slide show.
slideShowEnd(presentationName)
    Occurs after a slide show ends, immediately after the last SlideShowNextSlide event occurs. Returns name of presentation ending slide show.
slideShowNextBuild(presentationName)
    Occurs upon mouse-click or timing animation, but before the animated object becomes visible. Returns name of presentation where event occurred.
slideShowNextClick(presentationName)
    Occurs on the next click of the slide. Returns name of presentation containing clicked slide.
slideShowNextSlide(presentationName)
    Occurs immediately before the transition to the next slide. For the first slide, occurs immediately after the SlideShowBegin event. Returns name of presentationcontaining transitioning slides.
presentationBeforeSave(presentationName)
    Occurs before a presentation is saved. Returns name of presentation being saved.
presentationClose(presentationName)
    Occurs immediately before any open presentation closes. Returns name of presentation closing.
presentationNewSlide(slideIndex)
    Occurs when a new slide is created in any open presentation. Returns index of new slide.
presentationOpen(presentationName)
    Occurs after an existing presentation is opened. Returns name of presetation opened.
presentationPrint(presentationName)
    Occurs before a presentation is printed. Returns name of presentation to be printed.
presentationSave(presentationName)
    Occurs before any open presentation is saved. Returns name of presentation being saved.
windowActivate(presentationName)
    Occurs when the application window or any document window is activated. Returns name of presentation containing the activation event.
windowDeactivate(presentationName)
    Occurs when the application window or any document window is deactivated. Returns name of presentation containing the deactivation event.
