Event Handling


Events are captured in the Excel application and information is passed to MarkLogicExcelEventSupport.js which in turn calls the functions in MarkLogicExcelEventHandlers.js. Developers should define their custom functions within MarkLogicEventHandlers.js

Following events captured:

MLA.chartObjectMouseDown()
MLA.sheetActivate()
MLA.sheetBeforeDoubleClick()
MLA.sheetBeforeRightClick()
MLA.sheetChange()
MLA.sheetDeactivate()
MLA.rangeSelected()
-sheetSelectionChange Event only caught when selection is named range
MLA.workbookActivate()
MLA.workbookAfterXmlExport()
MLA.workbookAfterXmlImport()
MLA.workbookBeforeXmlExport()
MLA.workbookBeforeXmlImport()
MLA.workbookBeforeClose()
MLA.workbookBeforeSave()
MLA.workbookDeactivate()
MLA.workbookNewSheet()
MLA.workbookOpen()

Event Detail

MLA.chartObjectMouseDownHandler(chartName)
    Occurs when a mouse button is pressed while the pointer is over a chart. Returns name of chart selected.
    Note: This is not an application level event, see MLA.addChartObjectMouseDownEvents()
MLA.rangeSelectedHandler(rangeName)
    Occurs when the named range is selected on any worksheet. Returns named ranges selected.
MLA.sheetActivateHandler(sheetName)
    Occurs when any sheet is activated. Returns name of sheet activated.
MLA.sheetBeforeDoubleClickHandler(sheetName, range)
    Occurs when any worksheet is double-clicked, before the default double-click action. Returns sheetName and range where double-click is about to occur.
MLA.sheetBeforeRightClickHandler(sheetName, range)
    Occurs when any worksheet is right-clicked, before the default right-click action. Returns sheetName and range where right-click is about to occur.
MLA.sheetChangeHandler(range)
    Occurs when cells in any worksheet are changed by the user or by an external link. Returns range where change occurred.
MLA.sheetDeactivateHandler(sheetName)
    Occurs when any sheet is deactivated. Returns name of sheet deactivated.
MLA.workbookActivateHandler(workbookName)
    Occurs when any workbook is activated. Returns name of workbook activated.
MLA.workbookAfterXmlExportHandler(workbookName, mapName, url)
    Occurs after Microsoft Excel saves or exports XML data from the specified workbook. Returns the following

    workbookName: name of workbook that exported XML
    mapName: the name of the XmlMap used to export the data
    url: the location of the exported file
MLA.workbookAfterXmlImportHandler(workbookName, mapName, refresh)
    Occurs after Excel saves or exports XML data from the specified workbook. Returns the following

    workbookName: name of workbook that imported XML
    mapName: the name of the XmlMap used to import the data
    refresh: "true" if the event was triggered by refreshing an existing connection to XML data, "false" if a new mapping was created.
MLA.workbookBeforeCloseHandler(workbookName)
    Occurs immediately before any open workbook closes. Returns name of workbook closing.
MLA.workbookBeforeSaveHandler(workbookName)
    Occurs before any open workbook is saved. Returns name of workbook saving.
MLA.workbookBeforeXmlExportHandler(workbookName, mapName, url)
    Occurs before Excel saves or exports XML data from the specified workbook. Returns the following

    workbookName: name of workbook that is exporting XML
    mapName: the name of the XmlMap to be used to export the data
    url: the location to export the file to
MLA.workbookBeforeXmlImportHandler(workbookName, mapName, refresh)
    Occurs after Excel saves or exports XML data from the specified workbook. Returns the following

    workbookName: name of workbook that is importing XML
    mapName: the name of the XmlMap used to be used to import the data
    refresh: "true" if the event was triggered by refreshing an existing connection to XML data, "false" if a new mapping was created.
MLA.workbookDeactivateHandler(workbookName)
    Occurs when any workbook is deactivated. Returns name of workbook deactivated.
MLA.workbookNewSheetHandler(workbookName, sheetName)
    Occurs when a new sheet is created in any open workbook. Returns name of workbook adding sheet and name of sheet added.
MLA.workbookOpenHandler(workbookName)
    Occurs when a workbook is opened. Returns name of workbook opened.
